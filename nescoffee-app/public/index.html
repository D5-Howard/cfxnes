NESCoffee

A Nintendo Entertainment System emulator written in CoffeeScript.

NESCoffee is in early development, so it still isn't working correctly (only few games are playable). The overall performance is quite slow (at least 1.7 GHz CPU and the latest Google Chrome are recommended). Try out JSNES as a better alternative. The source code is available at bitbucket.
